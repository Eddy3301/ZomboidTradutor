# ZomboidTradutor
Automação simples que visa traduzir todos os elementos do jogo Project Zomboid, visto que a tradução nativa do jogo embaralha elementos e deixa o jogo 50% em português e 50% em inglês

# Como usar 
Pegue o arquivo ItemName.json da pasta SAIDA, e o cole aqui: 

Caminho até sua steam\Steam\steamapps\common\ProjectZomboid\media\lua\shared\Translate\PTBR

Novas atualizações corrigiram os outros arquivos.
